Class Overview

Base class for the Plugin Engine. Classes implementing the engine should provide their own plugin interface based on Plugin and extend the Plugins.Impl with implementation of their own operations.

Plugins of such engines should implement that interface and must put Plugin-Class entry into the manifest file, e.g.: 

Plugin-Class: reflect.android.plugin.AndroidPlugin

Plugin object is created only, if the Jar file is placed in the directory expected by the Plugins Engine, manifest entry is present and the class pointed by this entry implements the interface of this particular Plugins Engine.

If Jar placed in the plugins directory is not a plugin, it is treated as an exceptional situation.
