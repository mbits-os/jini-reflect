Class Overview

Adds support for the Android NDK's AndroidBitmapInfo and API to lock and unlock pixmap inside the Bitmap.

To lock/unlock and access the pixmap and info: 

 Bitmap bm = ...
 BitmapLockInfo bmi = bm.lock();
 if (bmi.m_pixels != NULL) {
     ... = bmi.m_info.width;
     //...
     bm.unlock();
 }
